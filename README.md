# Air Quality Test

My solution for [Air Quality code test](https://github.com/bbc/vj-code-tests/tree/master/airquality), built with [Gridsome](https://gridsome.org/).

To develop clone this repository, enter the directory and run `yarn` to install required packages.

Run:
* `yarn develop` to run locally.
* `yarn build` to build
* `yarn explore` to view GraphQL data# wokorach-air-quality
